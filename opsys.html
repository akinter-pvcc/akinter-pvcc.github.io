Operating Systems Terminology

TERM DEFINITION
operating system the software that controls the overall operation of a computer
job queue where the jobs residing in mass storage wait for execution
FIFO first-in, first-out removal order for a queue
interactive processing operating systems that allow a program to being executed to carry on a dialogue with the user through remote terminals
time-sharing operating systems that provide service to multiple users at the same time
multiprogramming time is divided into intervals and then executtion of each job is restricted to only one interval at a time
application software programs for performing tasks particular to the machine's utilization
system software programs for performing tasks that are common to computer systems in general
utility software units that exented the capabilities of the operating system
graphical user interface (GUI) modern user interface that communicates with users through images and icons
shell older user interfaces that communicated with users through textual messages using keyboard and monitor screen
kernel the internal part of an operating system that contains software components that performt the basic functions of the computer
device driver software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
file manager coordinates the use of the machine's mass storage facilities
memory manager coordinates the machine's use of main memory
virtual manager a large "fictional" memory space created by paging
boot strapping a process performed by a computer each time it is turned on
ROM a small portion of a computer's main memory where the CPU expects to find its initial program
deadlock the condition in which two or more processes are blocked from progressing because each is waiting for a resrouce that is allocated to another
spooling a technique of holding data for output at a later but more convenient time
super user (administrator) account with highly privileged access to the operating system
multi-factor authentication a series of different authentications to prevent unathorized access
