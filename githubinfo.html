All About Github

  • What is a version control system?
  • A. A version control system is a sytem that tracks and managages changes to software code.
  • What is Git?
  • A. Git is free and open source distributed version control system used for coordinating work among programmers.
  • What is GitHub?
  • A. GitHub is an Internet hosting service for software development and control using Git.
  • What is the difference between Git and GitHub?
  • A. Git is a tool used to manage multiple versions of code edits that are transfered to a repository, while GitHub is a location for uploading copies of a Git repository.
  • Who started GitHub and how was it started?
  • A. GitHub.com was launched in April 2008 by Tom Preston-Werner, Chris Wanstrath, PJ Hyett, and Scott Chacon
  • What company owns it now?
  • A. GitHub, Inc. is currently owned by Microsoft.
  • How much does a GitHub account cost?
  • A. GitHub accounts are free, but there are GitHub subscription services.
  • What is the Octocat?
  • A. Octocat is a tamed ghost who is a cross between an octopus and a cat.
